Project Overview

RJN Child Minding Service is a website created to establish the online presence of a new childminding business. The website is designed to provide essential information about the childminding services offered, including the businessâ€™s philosophy, contact details, qualifications, and testimonials. It serves as an informational hub for parents looking for reliable and professional childminding services, making it easy to find out more about RJN's services and get in touch with the business.

I structured the entire website using clean, semantic HTML5. This ensured that the site was accessible, SEO-friendly, and easy to maintain. The pages included sections such as About Us, Services Offered, Contact Details, and Parent Testimonials. I created a custom design for the website using CSS and SCSS. SCSS (Sass) allowed for modular styling and better organization of the stylesheet, making it easier to manage and update the design as needed.I implemented a child-friendly color palette and font style to convey trustworthiness and professionalism while maintaining a warm, welcoming atmosphere. Using CSS Flexbox and Grid, I ensured that the layout adapted smoothly to different screen sizes and devices, providing a consistent experience across desktops, tablets, and mobile phones
